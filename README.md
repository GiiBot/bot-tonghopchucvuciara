## Crew Report Discord Bot

### Tính năng
- /crew_report (phân trang, UI đẹp)
- Thống kê theo role chức vụ
- Online / Offline / Inactive
- Auto báo cáo hàng tuần
- Log sang channel riêng
- Giới hạn role được tính

### Railway
Add Variables:
- DISCORD_TOKEN
- GUILD_ID
- LOG_CHANNEL_ID
- CREW_ROLE_IDS
